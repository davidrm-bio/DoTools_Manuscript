# DoTools Manuscript

This repository contains the data and scripts used on the associated manuscript.

- [GenerateFigures](Scripts/GenerateFigures.py) contains the code used to generate the figures.
- [Figures](Figures) contains the individual figures.
- [Manuscript](Manuscript) contains the manuscript figures.
- [Objects](Objects) contains the objects used to generate the figures.
